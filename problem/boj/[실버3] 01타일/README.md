## 01 타일

|풀이 사이트|문제이름|난이도|
|:---:|:---:|:---:|
|[백준](https://www.acmicpc.net/problem/1904)|01 타일|실버 3|

### 문제 풀이

N의 값에 따른 만들 수 있는 개수의 값은 다음과 같다.

**N = 1** (1)
```
1
```

**N = 2** (2)
```
11, 00
```

**N = 3** (3)

```
111, 100, 001
```

**N = 4** (5)

```
1111, 1001, 0011, 1100, 0011
```

이를 바탕으로 DP 식을 구성해보면

```
dp[i] = dp[i-1] + dp[i-2]
```

와 같다. 이 DP 식과 문제 조건에 있는 mod를 적용해서 정답을 출력하면 된다.
